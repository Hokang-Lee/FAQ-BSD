Enterprise 版アンチウイルス機能のライセンスを確認する方法

Enterprise シリーズでのアンチウイルス機能が働くためには、fsupdd/fsengd サービスのライセンスが有効である必要があります。fsupdd/fsengd ライセンスは、次年度更新時のライセンスキー登録することで365日が加算される形で有効となりますが、有効期間がどのくらい残っているかを確認するには、次の操作を行ってください。 E-POSTコントロールセンターのメールサーバ管理から［システム管理メニュー］をクリック、［ライセンス情報］をクリックして表示される「ライセンス」画面で確認してください。「Anti-Virus server license」項目の残り日数を確認してください。
残り日数が"0"より大きければ「Anti-Virus server license」は“有効”ということになり、"0"になっていれば「Anti-Virus server license」期間が終わっていることを表します。
