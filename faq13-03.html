特定のSMTPサーバへの中継機能はありますか？

E-Post Mail Server V・E-Post SMTP Server V シリーズには、特定のSMTPサーバへ中継するSMTPゲートウェイ機能が備わっています。
DMZ上にSMTPゲートウェイとして設定したマシンを置き、DMZを介して、インターネットと内部のメールシステムとの接続を行いたい場合などに設定することができます。

また、大量の接続が予想されるため、負荷分散を目的としてインターネット上に複数のMXレコードに指定するSMTPサーバとして設置したりするケースにも使えます。

一方、内側に設置するメールシステムは、SMTPゲートウェイにしているマシンからのみ、外部からの受信を受け入れるように設定することが必要ですが、E-Post Mail Server V シリーズを内側に設置するメールシステムに使い、SMTPゲートウェイにしているマシンからのみ外部からの受信を受け入れる設定にすることが可能です。
